# Changelog

All notable changes to Neutron CSS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Initial project setup
- Complete CSS reset
- Variables system (colors, spacing, typography, shadows, transitions)
- Utility classes:
  - Flexbox utilities (display, direction, justify, align, gap)
  - Grid utilities (display, columns, responsive breakpoints)
  - Spacing utilities (margin, padding)
  - Sizing utilities (width, height, max/min)
  - Position utilities (relative, absolute, fixed, sticky)
  - Display utilities (block, inline, flex, grid, hidden)
- Component styles:
  - Buttons (30+ variants including primary, secondary, outline, ghost, sizes)
  - Cards (20+ variants including shadow, border, gradient, glass)
  - Forms (inputs, selects, checkboxes, radios, switches)
  - Alerts (success, error, warning, info, primary variants)
  - Modals (overlay, header, body, footer, sizes)
  - Navigation (navbar, dropdowns, breadcrumbs, pagination, tabs)
- Theme support:
  - Light theme (default)
  - Dark theme
- Animations:
  - Fade (in, out, up, down, left, right)
  - Scale (in, up, down)
  - Slide (in up, down, left, right)
  - Rotate, spin, pulse, bounce, shake, blink
  - Skeleton loading animation
  - Hover effects (grow, shrink, lift, glow)
- Documentation site structure
- Example pages for components
- Build system with minification support
